# Компонент Figure.Show для 1С-Битрикс

Компонент для отображения зеленого круга или красного квадрата.

## Установка

Скопируйте содержимое папки `local` в корень вашего сайта Битрикс.

## Использование

В визуальном редакторе, в режиме редактирования исходного кода, добавьте компонент:
```php
<?php
$APPLICATION->IncludeComponent(
    "test_task:figure.show",
    "",
    [
        "FIGURE_TYPE" => "square" // или "circle"
    ]
);
?>
Или же добавьте его через элементы интерфейса. Путь к компоненту в режиме редактирования: Тестовое задание -> Отображение фигур
